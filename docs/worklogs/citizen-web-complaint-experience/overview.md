# Citizen Web Complaint Experience

## Scope

Expose the existing owner-private complaint and accountability capabilities in Citizen Web without
creating a second workflow or widening database access.

## Implemented Outcome

- protected, paginated complaint history;
- complaint detail and immutable status timeline;
- safe routing/location and public government resolution/action presentation;
- policy-bound feedback and reopening;
- explicit mobile handoff when reopening requires new location-bound media;
- loading, empty, dependency-error, and not-found states with the exact signed-in account visible.

The browser never reads the private complaint schema directly and cannot select routing, actor,
resolution, policy, or workflow identifiers.
